# bike-sharing-analysis.
Analisis data peminjaman sepeda menggunakan Python untuk memahami pola pengguna berdasarkan waktu dan hari kerja.
## Deskripsi Proyek
Proyek ini merupakan analisis data penyewaan sepeda menggunakan Bike Sharing Dataset.
Analisis dilakukan untuk memahami pola permintaan penyewaan sepeda berdasarkan faktor
waktu, musim, kondisi cuaca, serta tipe pengguna.

## Dataset yang Digunakan
Dataset yang digunakan adalah **Bike Sharing Dataset** yang berisi data penyewaan sepeda
harian dan per jam dengan beberapa variabel utama, antara lain:
- season, mnth, yr
- hr
- weathersit, temp, atemp, hum, windspeed
- workingday, weekday, holiday
- casual, registered, cnt

Dataset disimpan dalam folder `data/`.
